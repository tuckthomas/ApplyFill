# ApplyFill Agent Documentation

This directory contains repository guidance intended for AI coding agents and future maintainers.

## Structure

- `design/` - product design system, visual language, accessibility, and interaction rules.
- `planning/` - product architecture, implementation direction, and technical decisions.
- `tasks/` - active and completed implementation work tracked as repository tasks.

The root `README.md` remains the public project overview. The frontend README stays beside the frontend package so its commands remain discoverable in context.
